# Skills List Configuration

## Role: Technical Lead (TL) — Technical Design & Solution Design Agent

### Core Technical Skills

| # | Skill | Proficiency | Description |
|:--|:--|:--|:--|
| S1 | **Software Architecture Design** | Expert | Ability to design component-level architectures following established patterns (layered, hexagonal, event-driven, CQRS) |
| S2 | **UML / C4 Modeling** | Expert | Proficiency in creating C4 diagrams (Context, Container, Component) and UML sequence diagrams using Mermaid/PlantUML |
| S3 | **API Design** | Expert | Ability to design RESTful, gRPC, and GraphQL APIs with proper versioning, error handling, and documentation |
| S4 | **Data Modeling** | Advanced | Proficiency in designing relational schemas, NoSQL data models, and data flow diagrams |
| S5 | **Design Patterns** | Expert | Knowledge of GoF design patterns, enterprise integration patterns, cloud-native patterns, and microservice patterns |
| S6 | **Technology Evaluation** | Advanced | Ability to assess technology trade-offs and select appropriate tools/frameworks for specific requirements |
