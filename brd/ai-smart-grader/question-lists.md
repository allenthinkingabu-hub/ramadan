### Question List #1 — Core Business & Scope
1. 系统支持哪些学科？（数学、语文、英语、物理、化学？还是全学科？）
2. 系统支持哪些学段？（小学、初中、高中、大学？）
3. AI 分析题目时，除了判断对错，还需要提供什么？（解题步骤、知识点讲解、难度评级、类似题推荐？）
4. 错题本的核心功能有哪些？（错题收集、分类、知识点标签、复习提醒、统计分析、薄弱点识别、个性化推荐练习？）
5. 游客（未登录用户）可以体验哪些功能？
6. 系统是否需要支持多语言（中文、英文等）？
7. 是否有商业模式的初步规划？（免费/付费/订阅制/广告？）
8. 老师的班级管理功能具体包含哪些？（布置作业、批量查看、学情报告、班级排名？）
9. 家长查看学习报告的具体内容？（每日/周/月报告、错题趋势、知识点掌握率？）
10. 管理后台除了 AI Prompt 配置，还需要哪些管理功能？（用户管理、内容审核、数据统计、系统监控？）

### Answered Summary #1
- Q1: 全学科
- Q2: 所有学段（小学、初中、高中、大学）
- Q3: 解题步骤讲解、知识点标注、难度评级、类似题推荐
- Q4: 错题收集、分类、知识点标签、复习提醒、统计分析、薄弱点识别、个性化推荐
- Q5: 可体验 AI 判题和错题本功能，需 Agent 设计增加用户粘性且操作不复杂的方案
- Q6: 多语言，中文 + 英语
- Q7: 免费/付费/订阅制/广告，需 Agent 深入思考商业模式
- Q8: 班级管理、分析学生错题情况、知识掌握报表
- Q9: 家长暂时只看学习报告
- Q10: 用户管理、权限管理、以及系统全部管理功能，需 Agent 全面设计
- 补充：所有次数限制（游客体验次数、免费版每日次数、会员次数等）均需后台可配置

- Q1: 支持所有主流图片格式（JPG/PNG/PDF等），支持多张一次上传
- Q2: 图片大小限制后台可配置
- Q3: 手写体和印刷体都支持
- Q4: 多模型可切换，后台可配置
- Q5: 全部认证方式（手机号/邮箱/微信/Google/Apple ID）
- Q6: GDPR + 中国数据合规
- Q7: 99.9% 可用性
- Q8: 需 Agent 分析预期用户规模
- Q9: 需 Agent 分析 AI 响应时间期望
- Q10: 全部支付渠道（微信/支付宝/Apple Pay/Google Pay/Stripe），支持扩展不同国家支付方式
- Q8 Agent分析: 冷启动期DAU 1K-5K → 增长期5K-50K → 规模化50K-500K → 成熟期500K-2M；初始容量按DAU 10万规划预留10倍扩容
- Q9 Agent分析: 全链路4-10s，目标P50≤5s/P90≤8s/P99≤15s；采用流式返回+智能路由优化
