# Delivery Playbook
Ordered tasks across all roles. Waves indicate parallel batches; complete Wave N before Wave N+1. Within a wave, follow the listed order; each task triggers the tasks shown in Next (multiple = parallel fan-out).

## How to Navigate
- Locate phase, then wave. Execute tasks in order inside the wave; when a task lists multiple Next items, trigger them in parallel.
- Move to the next wave after all tasks in the current wave are done.
- All tasks from source files are included; wording kept intact.
