## Matrix

| Task / Deliverable | Technical Lead (TL) | Project Manager (PM) | IT Architect (SA) | Business Analyst (BA) | Dev Lead (DL) | Product Owner (PO) |
|:---|:---:|:---:|:---:|:---:|:---:|:---:|
| Define vision scope & objectives | R | A | C | C | I | C |
| Gather upstream architect inputs | R | I | C | I | I | I |
| Formulate technical vision statement | R | I | C | I | C | C |
| Define guiding principles | R | I | C | I | C | I |
| Create business-technology alignment matrix | R | A | C | C | I | C |

| Identify risks & dependencies | R | I | C | I | C | I |

| Review & approve final report | C | A | R | I | C | C |

## Legend

- **R** = Responsible -- Does the work
- **A** = Accountable -- Approves / owns the outcome
- **C** = Consulted -- Provides input before decisions
- **I** = Informed -- Notified of outcomes
